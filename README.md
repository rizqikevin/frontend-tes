# HWM DASHBOARD

## Getting Started

### Prerequisites

- Node.js (v14+ recommended)
- npm or yarn

### Installation

```bash
git clone https://git.delameta.id/software/master/tracomm/hwm-dashboard.git
cd react-auth-dashboard-flow
npm install
# or
yarn install
```

### Running the App

```bash
npm start
# or
yarn start
```
